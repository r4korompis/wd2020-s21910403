CSS Units

hi saya Rangga, kali ini kita akan latihan CSS Units yang terdiri dari Absolute Lenghts dan Relative Lenghts.

Absolute Lenghts

Unit Description Example
cm centimeters text with centimeters
mm milimeters text with milimeters
in inches (1in = 96px = 2.54cm) text with inches
px pixels (1px = 1/96th of 1in) text with pixels
pt points (1pt = 1/72 of 1in) text with points
pc picas (1pc = 12pt) text with picas


Relative Lenghts

Unit Description Example
em Relative to the font-size of the element (2em means 2 times the size of the current font) text with em
ex Relative to the x-height of the current font (rarely used) text with ex
ch Relative to width of the "0" (zero) text with ch
rem Relative to font-size of the root element text with rem
vw Relative to 1% of the width of the viewport* text with vw
vh Relative to 1% of the height of the viewport* text with vh
vmin Relative to 1% of viewport's* smaller dimension text with vmin
vmax Relative to 1% of viewport's* larger dimension text with vmax
% Relative to the parent element text with %
